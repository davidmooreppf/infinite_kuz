# The Infinite Kuz

A single-serving website that imagines Kyle Kuzma's 2025-26 NBA season never
ended. Takes his real season-to-date line with the Milwaukee Bucks and
procedurally generates projected box scores — including plus/minus — for an
imaginary continuing schedule of home and away games against real NBA
opponents.

Everything after April 12, 2026 is clearly labeled as hypothetical.

Live site: https://infinite-kuz.vercel.app/
